Instructions for installing skins

There is one small thing you should know before you ask questions.

So to use skins with the BMW-Sauber 2007, you have to create 2 folders first. Go into rFactor/GameData/Vehicles/FBMW07/SAUBER and create a folder called "skins" which itself includes a folder called "misc"

Now got back to the Sauber-Folder and copy all dds-textures into this misc-folder. The only exception is the bmw.dds, which comes directly into the skins-folder.

Why all this? Without the copied textures of the original car, you won't be able to switch back to the Sauber-Textures from the menu. This is to avoid this.

Uninstallation

Just remove your textures from where you put them
