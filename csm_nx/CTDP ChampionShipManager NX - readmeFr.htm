CTDP ChampionShipManager NX

© 2010 CTDP - Version 2.3 - Date : 01 juin 2010

Description

C'est un petit outil qui permet d'ajouter plus facilement des saisons pour rFactor. Vous pouvez définir vos propres circuits et voitures pour le mode championnat. Vous pouvez ajouter un nombre divers et aléatoire de circuits pour la saison (dans la mesure où vous les avez installés). Les nouvelles saisons seront enregistrées sans avoir de Disparitès si vous jouez en ligne.

Amusez-vous bien!

Installation & démarrage

Vous pouvez installer le programme dans le répertoire de votre choix. Cela n'a pas d'importance s'il est installé dans le répertoire de rFactor ou pas. Après l'installation et le premier lancement, vous devez définir la langue que vous voulez utiliser et sélectionner le répertoire de rFactor

Désinstallation

  • allez dans le répertoire de CTDP ChampionShipManager NX et cliquez sur unins000 ou cliquez sur désinstaller dans votre menu démarrer
