CTDP Bahrain Race-2005 Track for rFactor

© 2006 CTDP - Version 1.0 - Date: 1 May 2006

Description

The Bahrain Circuit marks the first CTDP release for rFactor and brings you one of the newest and also most exciting GP circuits today. Dusty road and desert feeling for garantied.

  • Many new high detail 3d objects including the Main Bahrain tower and stands.
  • New quality high-res textures throughout (Note the grass in Bahrain is Astroturf and as such is uniform in colour / texture and does not have mud around the edge of the track - This is intentional)
  • Full multi-texture / spec and bump mapping (support in Dx8 and Dx9)
  • Track specific terrain variables
  • Track specific sky - if you want other skys to show for the track then add them to the Bahrain folder
  • New video for the large video screens around the track
  • Dynamic and static shadows on many objects (Note the pit garage shadow and main pit stands shadow were not made dynamic due to the FPS impact)
  • Night lighting
  • VisGroup support - reducing objects drawn with lower track detail setting in game (see end of readme for setting effects)

So have Fun!

Instructions

  • To install the CTDP Bahrain Race Track for rFactor you need about 40 megabyte free space on your harddisk and a clean install of rFactor.
  • Copy the Bahrain folder into the locations folder within rFactor directory
  • This track is built for rFactor Version 1.070. Problems may occur with newer/older versions of rFactor.
  • This track was built for use with Dx9 and Dx8. There is no support for Dx7. For best visuals run under Dx9

Uninstallation

To uninstall this Track remove the Bahrain Folder in your rFactor/Gamedata/Locations Path

Miscellaneous

VisGroup Settings and Effects:

When the ingame track detail is changed from 'Maximum' The following objects will be affected:

  • Max = All track items in front view ---- small detail items turned off in rear view
  • High = Items below turned off in main view ---- small detail items turned off in rear view
    • TVs and TV stands on Main pit building only
    • Pit railings
  • Med = Items below turned off in main view ---- small detail items turned off in rear view
    • TVs and TV stands on Main pit building
    • All other TVs
    • Pit railings
  • Low = Items below turned off in main view ---- small detail items turned off in rear view
    • TVs and TV stands on Main pit building
    • All other TVs
    • Pit railings
    • Thr roof of the main stand and back pit stand
    • Pit tents and shadows
